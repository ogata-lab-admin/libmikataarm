# libmikataarm

Library for Mikata Arm by ROBOTIS Japan


## Setup


```
$ git clone https://github.com/ogata-lab/libmikata
$ cd libmikata
$ git submodule update --init --recursive
$ mkdir build
$ cd build
$ cmake ../
$ make
```
